## 简介

[UI](https://github.com/renmcc/UI) 是[SA后台](https://github.com/renmcc/SA)前端解决方案，它基于 [vue](https://github.com/vuejs/vue) 和 [element-ui](https://github.com/ElemeFE/element)实现。它使用了最新的前端技术栈，内置了动态路由，权限验证，提炼了典型的业务模型，提供了丰富的功能组件。
